ASCOM Dome Driver (C#) FOR LifeRoof /FA-Duino Mega

This driver is built from the original for the FA-Duino // LifeRoof motor system for a roll-off roof. Apart from the standard commands to open, close, abort and detect the shutter state, it includes some custom controls. The open and close commands are only executed by the Arduino if it is safe to do so, that is, it does not hit the mount or it is not raining. It responds to open and close commands with quick shutterstatus update and also updates slewing property, which is a new direction from ASCOM team.

Commands:

"CLOSE" CloseRoof;

"OPEN" OpenRoof;

"INIT" Initialise Roof system (non-standard);

"ABORT" AbortRoof;

"FORCEOPEN" (non-standard);

"FORCECLOSE" (non-standard);

"NORAINSENSE" (non-standard, disable rain sensor);

"NOPARKSENSE" (non-standard, disable park sensor);

"RAINSENSE" (non-standard, enable rain sensor);

"PARKSENSE" (non-standard, enable park sensor);

“PARKSENSOR” (non-standard, show park sensor enable status)

“RAINSENSOR” (non-standard, show rain sensor enable status)

“BEEPON” (non-standard, enable beeper)

“BEEPOFF” (non-standard, disable beeper)

The Arduino transmits a status, including environment sensory information, of which rain and mount park confirmation are currently active.

Message is 11 chars long, with $and # at either end.

“$raining,roofstatus,parked,rainsense,parksense,beepstatus#”   rain, rainsense, parksense, beepstatus and parked are Booleans, roofstatus is enumerated:

int shutterOpen = 0;

int shutterClosed = 1;

int shutterOpening = 2;

int shutterClosing = 3;

int shutterError = 4;
